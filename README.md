# moribvndvs dotfiles


## Install

Running the following script will install Homebrew, chezmoi, and perform a 
chezmoi init

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/moribvndvs/dotfiles/main/install.sh)"
```
